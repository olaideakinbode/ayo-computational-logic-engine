# ayo-computational-logic-engine
This is a Python implementation of the Ayo (Mancala) game logic, focusing on state-space management and circular array algorithms.

📌 **Project Overview**
This repository contains a robust Python implementation of the Ayo game, a traditional board game of the Yoruba people of West Africa. Developed during a research internship at the African Language Technology Initiative (ALT-I), this project serves as a case study in computational ethnomathematics—the intersection of indigenous knowledge systems and algorithmic modeling.

The primary objective was to translate complex, orally-transmitted traditional rules into a formal, executable state-space model using Object-Oriented Programming (OOP) and Circular Array Logic.

🧠 **Technical Logic & Data Structures**
The engine focuses on high-fidelity simulation of game mechanics through:

Circular Array Sowing: Implementing non-linear seed distribution across a 14-slot array (12 playing cups + 2 scoring holes).

Recursive Capture Algorithms: A backtracking-style logic that validates and executes seed captures based on end-state conditions (the "2 or 3 seeds" rule).

State Management: Explicit handling of game phases, turn-toggling, and boundary condition validation.

🛠 **Features**
**Refactored for Python 3:** Updated from legacy Python 2 to modern standards with type-hinting and improved input handling.

**Modular Architecture:** Separated concerns between board configuration, sowing logic, and capture mechanics.

**State-Space Search Foundation:** Built to support future integration of AI agents (Minimax or Reinforcement Learning).
